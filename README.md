# NotScrabble
## Overview

NotScrabble is a console-based word game where players draw random letter tiles and form valid English words to score points.
This project focuses on applying core PF concepts, including modular programming, arrays, strings, loops, conditionals, and basic file handling.

## Features

- Random tile generation

- Word input and validation

- Scoring based on letter values

- Turn-based gameplay

- Clean and simple console interface
